## Descrição do problema:
#### Faça uma função em linguagem C que calcule o valor da raiz quadrada de um número real.
## Solução matemática
#### r = (r / 2) + (n / (2 * r))
## Português estruturado:
#
##### Função raizq(real n)
##### INÍCIO
##### 	inteiro x;
##### 	real r = n;
##### 	para x de 0 até 9 após incremente 1
##### 		r = (r / 2) + (n / (2 * r));
##### 	retorne r;
##### FIM
